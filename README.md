# DLF-Avenue-Unified-Mall-Management-Dashboard
**Daily Dashboard Report**

Daily Dashboard Report is a temporary business solution designed to help mall management access operational and commercial performance data faster, more clearly, and with less manual effort. Instead of relying only on static daily Excel sheets, this project converts reporting into an interactive dashboard that brings together mall footfall, parking performance, total sales, cinema metrics, outlet-level comparisons, and trade density in one place.

The dashboard is built for practical business use in environments where full backend integration is not yet available. To support this, it offers a hybrid ingestion model:
- manual Excel uploads from the dashboard
- automatic refresh from a watched folder when daily files are available

This makes the project immediately useful as a bridge solution for business teams, while also keeping it ready for future integration with enterprise systems such     as Pathfinder or internal reporting pipelines.

**Business Value**

- Reduces the time spent opening, searching, and comparing multiple Excel files
- Gives management a single-window view of daily mall performance
- Supports faster outlet-level decision-making through search and comparison
- Highlights business movement through trend charts, KPI summaries, and date-based analysis
- Creates a scalable foundation for future data automation

This project is suitable as a temporary or pilot reporting layer for mall operations teams, business analysts, leasing teams, and management stakeholders who need faster access to daily performance data before a full enterprise integration is implemented.
